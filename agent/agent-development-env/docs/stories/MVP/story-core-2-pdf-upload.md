# Story: Implement PDF Upload Component with Validation

## User Story

As a user, I want to upload PDF files via drag-and-drop or click so that I can submit documents for processing.

## Acceptance Criteria

- [ ] Drag-and-drop area
- [ ] Click to select files
- [ ] PDF file type validation
- [ ] File size limit (10MB)
- [ ] Error messages for invalid files

## Technical Details

- UploadZone component
- File API validation
- Visual feedback

## Definition of Done

- Upload component works
- Validation functions
- Errors displayed
